Iron Giant Garage (Minimum Requirements)

Run:
1) npm install
2) Create database and import: db/schema.sql
3) npm start
4) http://localhost:3000

Notes:
- Uses localStorage for "admin/secret" login and remembering last routine.
- 2 Web API fetch calls are in public/script.js (Quotable + Advice Slip).
- 4 views max: index, exercises, routines, history.
- All client JS is in public/script.js.
